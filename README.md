## Tmux Cheat Sheet

* Split pane horizontally:
  <br>`Prefix` `"`
  
* Split pane vertically:
  <br>`Prefix` `%`
  
* Create new window:
  <br>`Prefix` `c`
  
* Auto arrange panes:
  <br>`Prefix` `space`
  
* Auto re-arrange pane order:
  <br>`Prefix` `ctrl+o`

* Reorder pane down / up:
  <br>`Prefix` `{` / `Prefix` `}`
